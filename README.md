# 🚀 AI Resume Analyzer

An intelligent, full-stack AI-powered Resume Analyzer that helps users instantly evaluate resumes, match job descriptions, and improve ATS compatibility using modern Generative AI technologies.

---

## ✨ Features

- 📄 Upload resumes in seconds
- 🤖 AI-powered resume analysis
- 🎯 ATS score evaluation
- 🧠 Smart skill extraction
- 📊 Resume insights & recommendations
- 💼 Job description matching
- ⚡ Fast and responsive UI
- 🔐 Secure backend architecture
- 🌐 Full-stack MERN + AI integration

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- ⚡ Vite
- 🎨 Modern CSS / Tailwind CSS

### Backend
- 🟢 Node.js
- 🚂 Express.js

### AI Integration
- 🧠 Generative AI APIs
- 📑 NLP-based Resume Parsing

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Mayank87426/AI-Resume-Analyzer.git
```

---

### 2️⃣ Navigate to Project

```bash
cd AI-Resume-Analyzer
```

---

### 3️⃣ Install Backend Dependencies

```bash
cd Backend
npm install
```

---

### 4️⃣ Install Frontend Dependencies

```bash
cd ../Frontend
npm install
```

---

## 🔐 Environment Variables

Create a `.env` file inside the Backend folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_api_key
JWT_SECRET=your_secret
```

---

## ▶️ Run the Project

### Start Backend Server

```bash
cd Backend
npm run dev
```

---

### Start Frontend

```bash
cd Frontend
npm run dev
```
